# WishGalaxy Bot

This repository contains a very small example Telegram bot using
`python-telegram-bot`. Only placeholder features are implemented, but it can be
expanded to handle real business logic.

## Requirements

- Python 3.11
- Dependencies listed in `requirements.txt`

## Installation

1. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Set the `BOT_TOKEN` environment variable and run the bot:
   ```bash
   export BOT_TOKEN=<your-telegram-bot-token>
   python main.py
   ```

Alternatively you can execute `./start.sh` which runs the same command.

Feel free to modify `main.py` and add your own Telegram bot logic.
